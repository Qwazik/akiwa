Кнопки

Для стилизации используется основной класс btn и дополнительные


btn--default

<a class="btn btn--default">Кнопка</a>
Кнопка

btn--red

<a href="#" class="btn btn--red">Кнопка</a>
Кнопка

Кнопке с классом btn--red можно добавить класс big для получения большой кнопки

<a href="#" class="btn btn--red big">Кнопка</a>
Кнопка

btn--thin

<a href="#" class="btn btn--thin">Кнопка</a>
Кнопка

btn--brg

<a href="#" class="btn btn--brg">Кнопка</a>
Кнопка

Списки

Для стилизации используется основной класс ul(ненумерованные) ol(нумерованные) и дополнительные


просто ненумерованный список

<ul class="ul"> <li>Lorem.</li> <li>Velit.</li> <li>Cum.</li> </ul>
  • Lorem.
  • Velit.
  • Cum.

ul--circle

<ul class="ul ul--circle"> <li>Lorem.</li> <li>Velit.</li> <li>Cum.</li> </ul>
  • Lorem.
  • Velit.
  • Cum.

ul--check

<ul class="ul ul--check"> <li>Lorem.</li> <li>Velit.</li> <li>Cum.</li> </ul>
  • Lorem.
  • Velit.
  • Cum.

просто нумерованный список

<ol class="ol"> <li>Lorem.</li> <li>Velit.</li> <li>Cum.</li> </ol>
  1. Lorem.
  2. Velit.
  3. Cum.
